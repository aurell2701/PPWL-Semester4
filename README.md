# Jawaban Praktikum 2
1. Di Tailwind v3, struktur CSS masih menggunakan directive lama seperti @tailwind base;, @tailwind components;, dan @tailwind utilities; biar Tailwind bisa jalan. Tapi di Tailwind v4, udah lebih simpel, cukup pakai satu @import "tailwindcss";. Jadi v4 merupakan versi yang lebih ringkas dan modern.

2. Di Tailwind v3, kalau mau kasih warna transparan harus pakai dua class, misalnya bg-blue-500 dan bg-opacity-50. Tapi di Tailwind v4, bisa langsung ditulis jadi bg-blue-500/50. Jadi v4 merupakan versi kode lebih singkat dan cepat dipakai.

# Jawaban Praktikum 4
